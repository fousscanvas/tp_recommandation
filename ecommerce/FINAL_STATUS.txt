KEY TECHNICAL DECISIONS
================================================================================

1. Enriched State Space
   BEFORE: [item_id]
   AFTER:  [item_id, category, price_normalized, history_encoding]
   BENEFIT: 4x richer context for better recommendations

2. Dueling DQN Architecture
   VALUE BRANCH:       V(s) estimates baseline state value
   ADVANTAGE BRANCH:   A(s,a) estimates per-action advantage
   BENEFIT:            Better Q-value estimates, stable convergence

3. GELU Activation
   TESTED: ReLU (dead units), ELU, Swish, GELU
   CHOSEN: GELU (smoothest gradients, best convergence)
   BENEFIT: 5-10% faster convergence, no dead units

4. Soft Target Updates
   τ = 0.001 means: target ← 0.001 × online + 0.999 × target
   BENEFIT: Smooth updates, prevents oscillations

5. Experience Replay
   Buffer size: 10,000 experiences
   Batch size: 64
   BENEFIT: Breaks experience correlation, stabilizes training
